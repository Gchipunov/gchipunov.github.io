Step 1: Create a Shader Toy prototype

The first step is to create a new repository on GitHub. **Crucially**, you must name the repository exactly as your GitHub username, followed by `.github.io` (e.g., `yourusername.github.io`). This is how GitHub Pages automatically recognizes and deploys your site.

# The repository name is key!
yourusername.github.io

Step 2: Add HTML and CSS Files

Make core HLSL files and take a look at moddb tutorial:

  • **`get ready`**: Read a little about HLSL and Valves approach to inject it into the mess.
  • **`style.css`**: Your styling rules.
  • **`tutorial-one.html`**: The file you are currently reading (your tutorial content).

Ensure your `index.html` and other post pages link to the `style.css` file correctly in the `` section.

<link rel="stylesheet" href="style.css">
